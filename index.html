Toasts

You can use links or buttons to trigger toasts, as this boilerplate includes a small JS script to handle this for you.

To use a button add the attributes data-bs-toggle and data-bs-target to it.
<button class="btn btn-secondary" data-bs-toggle="toast" data-bs-target"#demoToastOne">...</button>

To use a link add the attributes data-bs-toggle and href to it.
<a href="#demoToastOne" data-bs-toggle="toast">...</a>
